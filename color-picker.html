Guide

What is a HEX Color Code?

A HEX (Hexadecimal) color code is a 6-symbol code made of up to three 2-symbol elements. Each of the 2-symbol elements expresses a color value from 00 to FF representing Red, Green, and Blue values respectively. For example, #000000 is absolute Black, and #FFFFFF is pure White.

Web developers and UI/UX designers use these codes inside CSS files to apply exact, consistent colors across websites and mobile applications.

FAQs

What is the difference between HEX and RGB?

HEX is base-16 math, while RGB uses traditional base-10 math (from 0 to 255). They both represent the exact same colors on digital screens, but HEX is shorter and more commonly used in CSS.
